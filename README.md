## 👀 预览

[查看文档](https://buzzts.netlify.app)

## 📡 安装依赖


```bash
# npm
npm install buzzts

# yarn
yarn add buzzts

# pnpm 🔥
pnpm add buzzts
```

## 🚀 运行

```bash
yarn dev
```

## 🕋 打包

```bash
yarn build
```

> 如果你不想在项目中引入太多依赖，而又想使用某一个方法
> 那么可以复制文档中的源码，在你的项目中引入
